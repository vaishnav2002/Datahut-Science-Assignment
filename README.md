# Nike Women Shoes Scraper

This is a Python script that scrapes product data from the **Nike India – Women’s Shoes** page using Playwright and saves the data to a CSV file.

---

## What this script does

* Loads the Nike women’s shoes page
* Scrolls to load all products
* Extracts product details
* Saves the data into a CSV file

---

## Data Collected

* Product Name
* Category
* Price
* Product URL
* Gender
* Has Price
* Scraped Date

---

## Requirements

* Python 3
* Playwright
* Pandas

---

## Installation

```bash
pip install playwright pandas
playwright install
```

---

## How to Run

```bash
python scraper.py
```

---

## Output

* `nike_womens_shoes_final.csv`
